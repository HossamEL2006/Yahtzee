# Yahtzee Game in Pygame
### (This is a very old project that I made back when I was learning Pygame)
This is a Yahtzee game implemented using Pygame. The game features dice rolling, scoring, and a user interface for selecting scores. It includes the classic Yahtzee scoring categories and a graphical representation of dice rolls.

Features
- **Dice Rolling**: Roll up to 5 dice with visual representation.
- **Scoring Categories**: Implemented all standard Yahtzee scoring categories.
- **User Interface**: Interactive buttons for rolling dice and marking scores.
- **Graphics**: Custom dice images and fonts.

## Requirements
- Python 3.x
- Pygame library
